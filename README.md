# Self-Organized Criticality in Social Media Information Cascades

This repository contains the Python simulation code and related documents for analyzing information cascades as a complex system exhibiting Self-Organized Criticality (SOC). 

## Project Overview
This simulation models users as nodes in a Scale-Free (Barabási–Albert) network. It explores how stochastic "noise" (spontaneous posting) propagates through the network to create "avalanches" (viral cascades).

## Files Included
* `soc_simulation.py`: The main Python script containing the NetworkX simulation.
* `avalanche_distribution.png`: Log-log plot demonstrating the power-law distribution of cascade sizes.
* `CLL_798_Individual_Project.pdf`: The final manuscript detailing the complexity analysis.

## How to Run
Run the simulation script from your terminal:
`python soc_simulation.py`
